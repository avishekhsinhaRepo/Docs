Helpful Resources

Open PDFs in Chrome

  1. Open Chrome

  2. Go to: chrome://settings/content

    1. Scroll to the bottom and open Additional Content Settings
    2. Click on PDF Documents
    3. Disable Download PDF Files

    Note: Chrome can only open local PDFs in a local HTML file in the browser and not an external application

Open PDFs outside of Firefox

  1. Open Firefox
  2. Go to about:preferences#general
  3. In the top right, Search for Applications
  4. Change Portable Document Format (PDF) to Use the Default application
